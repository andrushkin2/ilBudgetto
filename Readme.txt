Руководство по эксплуатации приложения "il Budgetto"

Установка приложения:

1)  Открыть дистрибутив ilBudgetto.exe, кливнув два раза
2)  Пройти все этапы мастера установки программ
3)  Открыть консоль
4)  Перейти в папку с утановленным приложением использую команду "cd pathToProgram"
    (чаще всего: "C:\Program Files\ilBudgetto")
5)  Открыть бразузер
6)  Зайти на сайт https://nodejs/org
7)  Скачать версию NodeJS версии 8.8 и выше
8)  Выполнить установку NodeJS используя местар установки программ
    (обязательно указать установку менеджера пакетов NPM)
9)  Открытой консоли выполнить команду: node -v
    (проверка установленного NodeJS).
    Если на консоли появится сообщение об ошибке - перезапустить консоль и попробовать еще раз
10) Выполнить команду: npm install -g tslint typescript gulp
    (должна пройти установка модулей)
11) Выполниуть команду: npm install .
    (будут установлены пакеты, необходимые для компиляции и запуска приложения)
12) Выполнить команду: tsc -w
    (должна пройти компиляция TS-файлов в JS-файлы)
13) Выполнить команду: gulp less
    (должны пройти компиляция LESS-файлов в CSS-файлы)
14) Выполнить команду: gulp build
    (должны выполниться сборка всех JS-файлов приложения в один исполняемый)
15) Выполнить команду: node server/server.JS
    (должно появится сообщение об открытом HTTP соединения и подключении к базе данных)
16) Открыть бразузер
17) Зайти на страницу: https://127.0.0.1
    (должно открытся приложения ilBudgetto)
